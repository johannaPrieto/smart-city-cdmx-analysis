# UrbanSignal CDMX

Infraestructura, Desorden y Delito: Análisis Espacial e Inteligente de la Relación entre Fallas de Servicios Urbanos e Incidencia Delictiva en la Ciudad de México.

## Objetivo
Analizar la relación entre reportes ciudadanos de infraestructura urbana e incidencia delictiva en la Ciudad de México mediante técnicas de análisis geoespacial, ciencia de datos e inteligencia artificial.

## Datasets principales
- FGJ CDMX — Carpetas de investigación
- Locatel 0311
- Colonias CDMX GeoJSON
- DENUE INEGI
- INEGI Población

## Herramientas
- Python
- Pandas
- GeoPandas
- Folium
- Scikit-learn
- Plotly
- Jupyter Notebook
